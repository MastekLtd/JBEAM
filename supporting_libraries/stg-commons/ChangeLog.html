Change Log

 

Release ${project.version}

Dated: ${timestamp}

 

 

Enhancements

 

1.     Day API has undergone changes and methods such as year(), month(), day(), hour(), minutes(), seconds(), have been changed to getYear(), getMonth(), getDay(), getHour(), getMinutes(), getSeconds().

2.     Helper classes that help generate MessageDigest have been added in crypto package. MD5 and SHA1 helper methods have been provided and the API is extensible to create any type of message digest A IODigestUitlity is added to handle large input streams.

 

Release 1.4

Dated: 20101206

 

 

Enhancements

 

1.      Introduced new class StringUtils. This class exhibits utility methods for manipulating string. The methods exhibited are split, join, countTokens, extractTokens and listTokens. These methods accept separator character as well as escape character. For all other utilities refer to the org.apache.commons.lang.StringUtils class.

 

Release 1.3

Dated: 20101206

 

 

Enhancements

 

1.     Introduced new methods in Day.

2.     RandomStringGenerator now works on constants example: Format is aAaa{a}999#{z} then a and z within brace brackets will not be replaced but kept as is in the generated string. Also an escape character is defined as /. Thus a open or close brace bracket can be added in the string as /{ or /} or // to append /.
