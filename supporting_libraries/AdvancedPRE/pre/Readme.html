Process Request Engine commonly referred to as PRE helps to execute tasks in an offline manner and has an inbuilt job scheduler.

 

The installation contains the following directory structure:

 

doc/

Refer to this directory for the API documentation.

 

lib/

All the libraries used by PRE reside here. Also the shell scripts associated automatically adds all the *.zip or *.jar files in the java classpath. You can add your own library files here in this folder.

 

properties/

PRE configuration files reside here. Refer to the PRE.pdf for further documentation of the property files and the meaning of each property within.

 

You can add your own configuration files here. It is recommended that the PRE property files should not be used for your own implementation instead you should create your own property files.

 

examples

Example classes for reference are located at this URL. This zip also contains some SQL files for inserting records to execute the example classes as well as it contains the table structure required to store the queue of PRE. These are specific to MYSQL. One can change these in accordance to one’s database. These databases include [Oracle, MSSQL, DB2 apart from MySQL] but are not limited to these. In general. the queries fired through PRE are in ANSI SQL and can be executed on any database (though not tested). The queries also can be configured through the pr.properties and jdbctype.properties. Please note that these examples are provide mearly from academic purpose and may not be the ideal way of coding and/or may not be free of bugs.

 

Changelog.html

Defines the revision history or change log of all the releases done so far.

 

Findbugs<version-releasenumber>.html

The findbugs is executed on the jar file before obfuscation and on the source code. The findbugs is executed by the ant build script. All the warnings described in the report have been reviewed and identified as non-bugs.

 

Pre.pdf

Installation as well as general documentation for PRE. This gives insight of how the PRE works.

 

Pre.ppt

Presentation that gives a very high level feature list.

 

preV1.0Rxx.xx.xxx.<datetimestamp>.jar

The PRE jar file.

 

startEng.sh

Shell script for starting the engine. For Unix.

 

stopEng.sh

Shell script for stopping the engine. For Unix.

 

killEng.sh

Shell script to kill engine. For Unix.
